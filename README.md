# CamelliaCare 架空の介護派遣サービス LP

ポートフォリオ用に制作した架空の介護派遣サービス「CamelliaCare」のランディングページです。
介護派遣スタッフの募集と登録を促すLPを設定し、未経験者でも安心して応募できるよう、
温かみのあるデザインと、迷わず進める直感的な応募フローを重視して制作しています。

---

## プロジェクト概要
介護業界で働きたい若年層をターゲットとしたランディングページです。  
「未経験でも働ける」「高時給」「安心のサポート体制」といった魅力を分かりやすく整理し、応募につながる導線を設計しました。

---

## 担当範囲
- 情報設計（構成案作成・セクション整理）
- デザイン制作（Figma）
- コーディング（HTML / SCSS / JavaScript）
- レスポンシブ対応
- デザインシステム作成（カラー、余白、UIガイド）

---

## 工夫した点
- 未経験者が持つ不安要素に対して、先に安心材料を提示する構成にした
- スマートフォンでの閲覧を基準としたレイアウト設計
- CamelliaCare のブランドカラーを基に、落ち着いた印象になるよう UI を統一
- セクション構成を「結論 → 理由 → 行動喚起」の流れで整理
- SCSS（FLOCSS）を基礎にモジュール化し、メンテナンス性を高めた

---
